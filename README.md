# IT-Vorwissen & Lernstil Test

Eine interaktive Streamlit-App für die Bewertung von IT-Vorwissen und Lernstilen.

## Features
- IT-Vorwissen Test (18 Fragen)
- Lernstil-Analyse (9 Fragen)
- Automatische Auswertung mit Visualisierung
- PDF-Export der Ergebnisse
- Admin-Bereich für Trainer
- CSV-Export aller Teilnehmer-Daten

## Installation
```bash
pip install -r requirements.txt
```

## Nutzung
```bash
streamlit run "import streamlit as st.py"
```

## Admin-Zugang
Passwort in der App ändern (Standard: admin123)
